# TP2: Programación Dinámica


## Requisitos
No se usaron librerías externas de uso poco común, pero para que se pueda ejecutar en todas las máquinas:

```bash
virtualenv .venv
```

### Linux
```bash
source .venv/bin/activate
```

### Windows
```cmd
.venv\Scripts\activate
```


Para ambos sistemas:
```bash
pip install -r requirements.txt
```

## Uso

Correr para un caso particular
```bash
python3 tp2.py path/to/oraciones.txt path/to/diccionario.txt
```

Correr los tests
```bash
python3 tp2.py test
```

Correr un ejemplo
```bash
python3 tp2.py ejemplo
```
